Documentation

Type in the 3 or 4 digit class code for the INST class you are looking for and select your class when it appears. The next screen will show all of the most recent professors who have taught the course. Once you select a section, you will see a graph of the section's grade distribution.

By comparing average GPA and the number of students who chose to take a class Pass/Fail/Withdrawal over the last 3 semesters, we can obtain a decent snapshot of how the COVID-19 pandemic has affected the quality of a class overall, and can help students decide if they should sign up for that specific section.
